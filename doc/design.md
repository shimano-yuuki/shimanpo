## 🎭 ローディング画面

### アニメーション仕様
```
初回アクセス時のみ表示
├─ 0.0s: ロゴフェードイン (duration: 0.8s, ease-out)
├─ 0.5s: ローディングバー表示 (duration: 1.5s)
├─ 1.8s: ロゴスケールアウト (duration: 0.5s)
└─ 2.3s: メインコンテンツフェードイン (duration: 0.6s)
```

### デザイン要素
- 中央配置のブログロゴ/アイコン
- シンプルなプログレスバー（Primary色）
- 背景: グラデーション `linear-gradient(135deg, #2563EB 0%, #10B981 100%)`
- オーバーレイ透明度: 95%

---

## 🎯 ヘッダーデザイン

### 基本仕様
- 高さ: 64px
- 背景: `rgba(255, 255, 255, 0.8)` + backdrop-filter blur
- スクロール時: 影を追加 `box-shadow: 0 2px 8px rgba(0,0,0,0.1)`
- Sticky position

### ナビゲーションアニメーション
```
マウスホバー時:
├─ テキスト色変更: Primary色へ (transition: 0.3s)
├─ 下線アニメーション: 左から右へ展開 (duration: 0.3s)
└─ テキスト微小拡大: scale(1.05)
```

### タブ構成
- **TOP**: ホームアイコン + テキスト
- **About**: 情報アイコン + テキスト
- **Post**: 編集アイコン + テキスト（admin限定）
  - admin表示時: バッジ「Admin」を右上に配置

### レスポンシブ
- モバイル（< 768px）: ハンバーガーメニュー
  - 開閉アニメーション: slide-in (0.3s ease-out)

## 📄 TOP画面（記事一覧）

### レイアウト
- 最大幅: 1200px（中央寄せ）
- グリッド: 3カラム（デスクトップ）→ 2カラム（タブレット）→ 1カラム（モバイル）
- カード間隔: 24px

### 記事カード
```
構成:
├─ サムネイル画像（16:9）
├─ タグバッジ（Secondary色）
├─ 記事タイトル（2行まで表示）
├─ 記事概要（3行まで表示）
├─ 投稿日時 + 読了時間
└─ 著者情報
```

### カードアニメーション
- **初期表示**: Stagger animation（順次フェードイン、各0.1s遅延）
- **ホバー時**:
  - カード全体: `translateY(-8px)` + 影強化
  - サムネイル: `scale(1.05)`
  - transition: 0.3s ease-out

### その他の要素
- ページネーション: 下部中央配置
- フィルタリング: タグによる絞り込み（アニメーション付き）
- 検索バー: ヘッダー下に配置、フォーカス時拡大

## ℹ️ About画面

### デザイン
- シンプルな1カラムレイアウト
- 最大幅: 800px
- セクション区切り: 水平線（Primary色のグラデーション）

### アニメーション
- スクロールトリガー: セクションごとにフェードインスライド
  - `translateY(40px)` → `translateY(0)`
  - Intersection Observer使用

### コンテンツ
- プロフィール画像: 円形、ホバーで回転
- SNSリンク: アイコンホバーでバウンド
- スキルバッジ: タグクラウド形式、インタラクティブ

## ✍️ Post画面（admin限定）

### レイアウト
- 2ペイン構成（編集 / プレビュー）
- 画面分割: 50% / 50%
- 切り替えボタン: トグルスイッチ（モバイル用）

### エディタ部分
```
構成:
├─ ツールバー（Markdown記法補助）
├─ タイトル入力フィールド
├─ タグ入力（複数選択可）
├─ Markdownエディタ（Monaco Editor風）
└─ 投稿ボタン（Accent色）
```

### プレビュー部分
- リアルタイムレンダリング
- 実際の記事表示と同じスタイリング
- シンタックスハイライト対応

### モード切り替えアニメーション
```
切り替え時:
├─ フェードアウト (0.2s)
├─ スライドイン (0.3s, ease-out)
└─ コンテンツフェードイン (0.2s)
```

### 投稿ボタン
- ホバー時: 拡大 + 影強化
- クリック時: リップルエフェクト
- 成功時: チェックアイコンアニメーション

## 🎬 マイクロインタラクション

### ボタン
- Hover: `scale(1.02)` + 明度変更
- Active: `scale(0.98)`
- Ripple effect（マテリアルデザイン風）

### インプットフィールド
- フォーカス時: ボーダー色変更（Primary）
- ラベルアニメーション: 上方向へ移動

### トースト通知
- 右上からスライドイン
- 自動消去: 3秒後フェードアウト
- 成功: Secondary色、エラー: Error色

### スクロールインジケーター
- ページトップに戻るボタン
- スクロール量に応じて表示/非表示
- クリックでスムーススクロール
